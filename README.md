# STM32F446 Bare-Metal Project (CMSIS only)

Bare-metal STM32F446 project using:
- STM32CubeIDE 2.0.0
- arm-none-eabi-gcc
- CMSIS (no HAL, no CubeMX)

## Features
- Custom startup and linker script
- GPIO control via registers
- SysTick without HAL

## Tested on
- NUCLEO-F446RE

## Build
Open the project in STM32CubeIDE and build normally.
